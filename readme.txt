Numpad controls:
  8 - rotate up
  2 - rotate down
  4 - rotate left
  6 - rotate right
  5 - switch projection
  / - previous mesh
  * - next mesh
  - - zoom out
  + - zoom in
  e - reset rotation

Keyboard controls:
  w - rotate up
  s - rotate down
  a - rotate left
  d - rotate right
  e - switch projection
  p - previous mesh
  n - next mesh
  x - zoom out
  z - zoom in
  r - reset rotation
